# Calculator App

A simple, responsive calculator built with plain HTML, CSS, and JavaScript.

## Live Demo

- [https://calculator-lovat-six-75.vercel.app](https://calculator-lovat-six-75.vercel.app)

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Decimal input support
- Percentage conversion (`%`)
- Clear (`C`) and delete/backspace (`DEL`)
- Keyboard input support
- Divide-by-zero error handling

## Project Structure

- `index.html` - Calculator layout and button markup
- `style.css` - Visual styling and responsive layout
- `script.js` - Calculator state management and logic

## Getting Started

1. Clone this repository:
   - `git clone https://github.com/AkashChaudhary14/calculator.git`
2. Open the project folder:
   - `cd calculator`
3. Open `index.html` in your browser.

No build step or dependencies are required.

## Keyboard Shortcuts

- `0-9` - Enter digits
- `.` - Decimal point
- `+`, `-`, `*`, `/` - Operators
- `Enter` or `=` - Calculate result
- `Backspace` - Delete last character
- `Escape` - Clear all

## Notes

- This app is fully client-side and runs directly in the browser.
- Results are rounded to avoid common floating-point display noise.
